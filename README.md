
# 🤖 Deeksha's Chatbot (Python Tkinter GUI)

A simple and interactive rule-based chatbot built using Python and Tkinter with a beautiful graphical user interface.

---

## 🚀 Features

- 💬 Interactive chatbot with predefined responses  
- 🎨 Beautiful GUI with colors and emojis  
- ⌨️ Input box with Enter key support  
- 🎲 Random replies for unknown inputs  
- 📜 Scrollable chat window  

---

## 🛠️ Technologies Used

- Python  
- Tkinter  
- Random Module  

---

## ▶️ How to Run

```bash
python chatbot.py
